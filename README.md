# Flutter + LPR API ナンバープレート認識サンプコード

ナンバープレート撮影記録用のAndroidアプリ『LPRMemo』で利用されている撮影した画像を、ナンバープレート認識API『LPR API』に
送り受け取った認識結果を表示するサンプルコードになります。

ご利用方法につきましては下記URLのブログ記事よりご確認をお願いします。
